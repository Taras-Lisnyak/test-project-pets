## ğŸ¶ About the Project

The website addresses an important social issue by simplifying the animal
adoption process. Users can:

- Meet the shelter residents online.
- Learn about their personalities and characteristics.
- Take the first step toward bringing an animal home.

The goal of the project is to make finding a new friend as simple, transparent,
and enjoyable as possible.

## âœ¨ Features

- **Interactive Gallery:** Browse all animals currently in the shelter.
- **Detailed Profiles:** Cards with photos, descriptions, history, and important
  characteristics of each animal.
- **Shelter History:** Short stories with photos; uses `Swiper` for viewing.
- **Reviews and Ratings:** Review system with rating visualization via
  `raty-js`.
- **Online Adoption:** Contact form to submit an adoption application directly
  through the website.
- **User-friendly UI/UX:** Accordions for compact information display and modals
  via `SweetAlert2`.
